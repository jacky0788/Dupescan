> 快速找出並清理磁碟中的重複檔案，釋放硬碟空間。

## 功能特色

- 掃描任意資料夾或磁碟機，找出完全相同的重複檔案
- 顯示每個重複群組的 Hash 值、檔案路徑、大小、修改時間
- 支援多種快速選取方式（保留最新、保留最舊、保留第一個）
- 一鍵刪除已勾選的重複檔案
- 顯示浪費空間統計
- **步驟進度顯示**：即時顯示目前執行到哪個步驟（步驟 X/3）及預估剩餘時間
- **暫停 / 繼續**：掃描中可隨時暫停，不占用 CPU/磁碟，繼續後從原點接續
- **取消掃描**：掃描按鈕切換為「取消掃描」，安全地中止目前任務
- **重置**：一鍵清空所有狀態，不會被任何任務卡死
- **自動 Log**：每次執行自動產生 `logs/dupescan_YYYYMMDD_HHMMSS.log`，記錄重要操作
- **群組排序**：可依群組大小、檔案數量、副檔名排序顯示結果
- **副檔名篩選**：只顯示特定副檔名的群組（如 `jpg,png`），支援多個副檔名同時篩選
- **批量快速選取**：一鍵對所有群組套用「保留最新/最舊/路徑最短/目錄最深/字母序…」等 9 種策略，大幅加快批量刪除流程
- **點擊開啟檔案**：點選藍色底線檔案名稱，以預設程式開啟
- **點擊路徑定位**：點選橘色底線路徑欄，Windows 檔案總管開啟並自動選取該檔案
- **圖片懸停預覽**：滑鼠移到圖片檔案名稱上，自動彈出縮圖預覽，移開後消失
- **📊 匯出分析報表**：掃描完成後，點擊「匯出報表」按鈕，以瀏覽器開啟 HTML 報表，包含：
  - 可釋放空間、群組數、重複檔案數等整體統計
  - 依副檔名繪製兩張 SVG 圓餅圖（浪費空間分布 / 檔案數量分布）
  - 完整副檔名明細表格（群組數、檔案數、總大小、可釋放空間、占比）
- **文字大小調整**：在「篩選與排序」面板中可調整文字大小（9–18px），縮小文字可讓每頁顯示更多群組
- **結果區固定**：結果捲動區固定佔滿下方空間，不受篩選面板展開/收起影響
- 深色主題，簡潔易用的 UI
- 3-Pass 優化演算法，大幅降低磁碟 I/O

## 安裝說明

### 1. 複製專案

```bash
git clone <your-repo-url>
cd dupescan
```

### 2. 安裝依賴

```bash
pip install -r requirements.txt
```

安裝的套件：

| 套件 | 用途 |
|------|------|
| `PyQt6` | 圖形介面框架 |
| `xxhash` | 高速檔案 Hash 演算法 |
| `humanize` | 人性化顯示檔案大小（如 1.2 MB） |

### 3. 啟動程式

```bash
python main.py
```

## 操作流程

### Step 1 — 選擇掃描路徑

點選上方「**瀏覽**」按鈕，選擇要掃描的資料夾或磁碟機根目錄（如 `C:\`）。

也可直接在輸入框中手動輸入路徑，例如：
```
D:\Photos
C:\Users\你的名字\Downloads
```

---

### Step 2 — 設定最小檔案大小（選用）

預設為 **1 KB**，會跳過小於此大小的檔案。

可依需求調整，例如設為 **100 KB** 以只掃描較大的檔案，加快掃描速度。

---

### Step 3 — 開始掃描

點選「**▶ 開始掃描**」按鈕，程式會依序執行三輪比對，並即時顯示步驟與預估剩餘時間：

```
步驟 1/3 — 依檔案大小分組（極快，無磁碟讀取）
步驟 2/3 — 快速 Hash 比對（前 4KB）     預估剩餘: 約 30 秒
步驟 3/3 — 完整 Hash 比對               預估剩餘: 約 8 秒
```

掃描中的操作選項：

| 按鈕 | 效果 |
|------|------|
| **⏸ 暫停** | 暫停掃描，不占用 CPU/磁碟，可隨時繼續 |
| **▶ 繼續** | 從暫停點接續掃描 |
| **✕ 取消掃描** | 中止目前任務，回到閒置狀態 |
| **🔄 重置** | 強制停止並清除所有結果，回到初始狀態 |

---

### Step 4 — 查看結果

掃描完成後，畫面會列出所有重複群組，並顯示篩選與排序工具列：

```
群組 #1  ·  3 個重複  ·  每個 2.4 MB        浪費 4.8 MB
─────────────────────────────────────────────────────────
Hash: a1b2c3d4e5f6...

快速選取: [保留最新] [保留最舊] [保留第一個] [全選] [全不選]

┌────┬────────────────┬──────────────────────────┬────────┬─────────────────┐
│ 刪除│ 檔案名稱       │ 路徑                     │ 大小   │ 修改時間        │
├────┼────────────────┼──────────────────────────┼────────┼─────────────────┤
│ ☐  │ photo_001.jpg  │ D:\Photos\2023           │ 2.4 MB │ 2023-06-15 14:22│
│ ☑  │ photo_001.jpg  │ D:\Photos\備份\2023      │ 2.4 MB │ 2023-06-15 14:22│
│ ☑  │ IMG_copy.jpg   │ C:\Users\Downloads       │ 2.4 MB │ 2023-08-01 09:10│
└────┴────────────────┴──────────────────────────┴────────┴─────────────────┘
```

頂部顯示：
- **X 個重複群組** — 找到的重複群組總數
- **浪費空間: X MB** — 可釋放的磁碟空間

#### 篩選與排序

掃描完成後，結果區上方會出現篩選/排序工具列：

| 控制項 | 說明 |
|--------|------|
| **副檔名篩選** | 輸入 `jpg` 或 `jpg,png` 只顯示符合副檔名的群組；留空顯示全部 |
| **排序依據** | 群組大小、檔案數量、副檔名（各有升降序）|

#### 批量快速選取

篩選/排序列下方有「批量快速選取」面板，一鍵對**所有顯示中的群組**同時套用策略：

| 按鈕 | 說明 |
|------|------|
| **修改最新** | 各群組保留修改時間最新的檔案，其餘勾選 |
| **修改最舊** | 各群組保留修改時間最舊的檔案，其餘勾選 |
| **路徑最短** | 各群組保留完整路徑字串最短的檔案 |
| **路徑最長** | 各群組保留完整路徑字串最長的檔案 |
| **目錄最淺** | 各群組保留目錄層數最少（最接近根目錄）的檔案 |
| **目錄最深** | 各群組保留目錄層數最多（最深層）的檔案 |
| **字母最前** | 各群組保留完整路徑字母序排最前的檔案 |
| **字母最後** | 各群組保留完整路徑字母序排最後的檔案 |
| **列表第一** | 各群組保留列表中第一筆 |
| **全部勾選** | 勾選所有群組的所有檔案（慎用！）|
| **全部取消** | 清除所有群組的勾選 |

每個群組卡片內也有相同的單群組快速選取按鈕，效果僅作用於該群組。

#### 點擊開啟檔案

點擊結果表格中的**藍色底線檔案名稱**，程式會以系統預設程式開啟該檔案。

#### 點擊路徑開啟資料夾

點擊**橘色底線路徑欄**，Windows 檔案總管會開啟並自動選取（highlight）對應的檔案。

#### 圖片懸停預覽

將滑鼠移到圖片檔案（.jpg、.png、.gif 等）的名稱上，會自動彈出縮圖預覽視窗；
滑鼠移開後預覽視窗自動關閉，無需點擊。

---

### Step 5 — 選擇要刪除的檔案

每個群組都有快速選取按鈕：

| 按鈕 | 效果 |
|------|------|
| **保留最新** | 勾選除了修改時間最新以外的所有檔案 |
| **保留最舊** | 勾選除了修改時間最舊以外的所有檔案 |
| **保留第一個** | 勾選第 2 筆以後的所有檔案 |
| **全選** | 勾選該群組所有檔案（慎用！） |
| **全不選** | 清除該群組的所有勾選 |

你也可以手動在每個檔案旁的勾選框逐一勾選。

---

### Step 6 — 刪除

確認勾選後，點選右上角「**🗑 刪除已勾選**」按鈕。

程式會彈出確認視窗，列出所有即將刪除的檔案，確認後才會執行刪除。

刪除完成後，程式會自動重新掃描以更新結果。

> **注意：** 刪除操作直接從磁碟移除檔案，不會放入資源回收筒，請謹慎操作。

## 掃描演算法

DupeScan 使用 **3-Pass 優化策略**，大幅減少不必要的磁碟讀取：

```
所有檔案 (N 個)
    │
    ▼ Pass 1：依大小分組（無 I/O）
    │  ── 跳過大小唯一的檔案
    │
    ▼ Pass 2：讀取前 4KB，快速 Hash
    │  ── 跳過前 4KB 不同的檔案（通常過濾 90% 以上）
    │
    ▼ Pass 3：對剩餘檔案做完整 xxHash
    │
    └─→ 確定重複群組
```

**Hash 演算法：**
- 優先使用 `xxHash`（比 MD5 快約 10 倍）
- 若未安裝，自動退回使用 Python 內建 `blake2b`

## 常見問題

**Q: 掃描完成後再按「開始掃描」會有問題嗎？**
A: 不會。程式採用掃描 ID 機制，確保前次執行緒完全清理後才啟動新掃描，不會發生閃退或狀態錯亂。

**Q: 掃描速度很慢？**
A: 建議提高「最小檔案大小」門檻，減少需比對的小檔案數量。掃描整顆磁碟（數十萬個檔案）可能需要數分鐘，屬正常現象。

**Q: 某些資料夾無法掃描？**
A: 系統保護資料夾（如 `C:\Windows`）可能需要系統管理員權限。請以「以系統管理員身份執行」的方式開啟終端機。

**Q: 刪除後能復原嗎？**
A: 目前版本直接刪除，不放入資源回收筒。建議刪除前先確認清楚，或手動將檔案移到資源回收筒。

**Q: 支援外接硬碟嗎？**
A: 支援。直接在路徑輸入外接硬碟的磁碟機代號（如 `E:\`）即可。
